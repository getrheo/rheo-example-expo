# rheo-example-expo

Runnable example for [`@getrheo/react-native-expo`](https://www.npmjs.com/package/%40getrheo%2Freact-native-expo).

**SDK repo:** [getrheo/rheo-react-native](https://github.com/getrheo/rheo-react-native)

```bash
cp .env.example .env
pnpm install
pnpm start
```
